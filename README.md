# **FalloutBadge**

![ProjectSnapshot](https://github.com/TXWD1234/FalloutBadge/raw/main/docs/readme-assets/ADD_PCB.png)

# Project Description
A simple half-adder circuit that calculate the result from addition of 2 one bit number.
The main challange in this project is to implement the logic gates, and use them to compose the adder.

# Logic Description
The adder have 2 input pins and 2 output pins:
- input: first adder number
- input: second adder number
- output: result (the value for the original digit)
  - this is a XOR gate
- output: carry (the additional result for the next digit)
  - this is an AND gate
